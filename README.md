# AI Chatbot (Basic)

A simple Python CLI chatbot using OpenAI API.

## Features
- Takes user input
- Sends to AI model
- Prints response

## Tech Stack
- Python
- OpenAI API

## How to run
1. Install dependencies:
   pip install openai python-dotenv

2. Add .env file:
   OPENAI_API_KEY=your_key

3. Run:
   python app.py
